使用VMware安装Ubuntu

发表于 2021-03-16 分类于 Linux
本文字数： 870 阅读时长 ≈ 1 分钟

安装VMware

下载

下载地址： Download VMware Workstation Pro

点击后进入，开始下载

安装

双击打开下载好的程序

点击下一步，开始安装步骤

接受条款并开始下一步

点击自动安装，开始下一步

选择安装路径，最好不要在系统盘，确认后开始下一步

勾选取消就ok，少点无用通知

点击下一步，安装即将完成

点击安装

等待一会，安装成功

激活

激活码(网上可搜到)

  • ZF3R0-FHED2-M80TY-8QYGC-NPKYF

  • ZF71R-DMX85-08DQY-8YMNC-PPHV8

  • YF390-0HF8P-M81RQ-2DXQE-M2UT6

    复制输入任意一个完成激活！

点击进入主页面

至此完成VMware安装

虚拟机內安装Ubuntu

下载Ubuntu镜像

点击下载：Download Ubuntu Desktop

新建虚拟机

打开安装好的VMware,进入主页面，点击创建新的虚拟机，进入新建虚拟机向导

选择自定义（高级），点击下一步

继续

选择稍后安装操作系统，点击下一步

我们要安装ubuntu,当然选择Linux了，ubuntu版本最好选择64位

选择合适的路径，最好不要在系统盘，名称自定

这一步看自己电脑配置，下一步会有提示

选择推荐的就ok

选择使用网络地址转换（NAT）

选择推荐的LSI Logic(L)就行

后续步骤如下：

磁盘设置20G就够了，也可以多一点

点击自定义硬件，进入新 CD/DVD(SATA)选项，点击使用ISO文件

选择你下载好的映像文件，点击关闭退出

最后点击完成虚拟机就创建好啦

安装Ubuntu

点击开启此虚拟机，即可开始安装

进入后稍等一会儿，开始安装

选择中文简体，点击安装Ubuntu

键盘布局自带英文，我们也可以在系统内下载中文输入法，这里选择Chinese

我们只是最为学习安装Ubuntu,上面选择最小安装即可.

其他选项将安装Ubuntu时下载更新取消，由于下载源在国外，选择后更新会很慢，这里在进入桌面后换一下源在更新就好了

要注意的是这里清理的是我们分配给虚拟机的20G磁盘，里面本来就是空的，清空无所谓

在这里设置自己的账户

开始安装，等待大约一个多小时，安装完成
